How It Works

Your keyboard is your controller.

Chord Runner uses the Web MIDI API to receive input directly from your electric piano or MIDI keyboard — no plugins, no drivers, no downloads.

Step 01
🎹

Connect Your Keyboard

Plug in any USB MIDI keyboard or electric piano. The game detects it automatically via the Web MIDI API — works in Chrome & Edge out of the box.

Step 02
🏃

Jump Over Obstacles

Chord blocks come flying at your character. Each block shows the chord you must play. Hit the right notes and your character leaps over it — miss and you lose a life.

Step 03
⚔️

Battle Monsters

Monsters appear mid-run. Enter combat mode: play the displayed chord within the time window to deal damage. Survive and defeat them to keep running.

Step 04
🏆

Score & Improve

Every correct chord earns points. Perfect timing gives bonus score. Build muscle memory for chord shapes while having fun — learning through play.

🎹

Designed for MIDI Keyboards

This game is built to be played with an electric piano or USB MIDI controller. Connect via USB to your computer, open the game in Chrome or Edge, and grant MIDI access when the browser asks. The game receives your notes in real time — no latency, no configuration needed.

No MIDI keyboard? No problem — use the on-screen virtual buttons to try the game anyway. Virtual mode is perfect for getting a feel of the gameplay before connecting a real keyboard.
